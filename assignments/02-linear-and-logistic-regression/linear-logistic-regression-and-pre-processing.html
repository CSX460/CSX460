The University of California, Irvine Machine Learning Repository. One popular data set is the German Credit Data. Using this data, create a logistic regression model.

data(GermanCredit)
set.seed(123456)
split = sample.split(GermanCredit$Class, SplitRatio = 0.7)
train = subset(GermanCredit, split==TRUE)
test = subset(GermanCredit, split==FALSE)
